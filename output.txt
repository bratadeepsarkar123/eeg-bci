EEG Brain Speller Project
Complete Requirements & Setup Guide for Mentees
Brain-Computer Interface (BCI) | Signal Processing | Machine Learning
1. Project Overview
You will build an EEG-based brain speller — a Brain-Computer Interface (BCI) system that allows a person to type characters using only their brain signals, with no physical movement. The system detects specific brainwave patterns in response to visual stimuli and uses machine learning to identify which letter the user intended.
The most common paradigm is the P300 Speller: a 6×6 matrix of characters flashes row by row and column by column. When the target character flashes, the brain produces a distinct EEG response called the P300 component (~300 ms after the flash). Your classifier detects this response to determine the intended character.

4. Signal Processing Pipeline
Implement each stage in order. Do not skip steps.
Stage 1 — Preprocessing
Bandpass filter: 0.1–30 Hz (P300) or 1–45 Hz (SSVEP/MI)
raw.filter(l_freq=0.1, h_freq=30.0)
Notch filter at 50 Hz (power line noise in India)
raw.notch_filter(freqs=50)
Re-reference to average or linked mastoids
raw.set_eeg_reference('average')
Bad channel interpolation (mark bad channels first)
Artifact rejection: use ICA to remove eye blinks and muscle artifacts
Stage 2 — Epoching
Extract epochs locked to stimulus onset events
Epoch window: −200 ms to +800 ms around each flash (for P300)
Apply baseline correction using the −200 to 0 ms pre-stimulus window
events, event_id = mne.events_from_annotations(raw)
epochs = mne.Epochs(raw, events, tmin=-0.2, tmax=0.8, baseline=(-0.2, 0))

Stage 4 — Classification
Baseline model: Linear Discriminant Analysis (LDA)
from sklearn.discriminant_analysis import LinearDiscriminantAnalysis
Strong classical baseline: SVM with RBF kernel
Deep learning: EEGNet (compact CNN for EEG) — use braindecode or implement from the 2018 paper
Ensemble: average classifier scores across multiple flash repetitions (improves accuracy significantly)
Stage 5 — Evaluation
Use stratified k-fold cross-validation (k=5 or k=10)
Report: accuracy, precision, recall, F1, and confusion matrix
Primary metric: Information Transfer Rate (ITR) in bits/minute
ITR Formula:
ITR = (log2(N) + P*log2(P) + (1-P)*log2((1-P)/(N-1))) * 60 / T
Where N = number of symbols (e.g. 36), P = classification accuracy (0–1), T = trial duration in seconds.
